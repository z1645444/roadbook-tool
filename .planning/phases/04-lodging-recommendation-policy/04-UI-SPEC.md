## Visual Hierarchy Contract

- Primary focal point: `Day N / overnight stop` heading and status badge (`compliant`, `relaxed`, `no-match`).
- Secondary focal point: per-category top result card (hostel, guesthouse, hotel).
- Tertiary information: fallback trace and policy stage labels.
- Icon-only actions are not used in this phase. If any icon is added later, include adjacent text label.
